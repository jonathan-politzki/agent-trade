# UI — Agent Trade visualizations

A static site that visualizes results from the used-car-salesman experiment.
No build step. Drop the analysis agent's output into `data/` and the views
re-render.

## Run

```bash
cd ui
python3 -m http.server 8765
# then open http://localhost:8765
```

Browsers block `file://` fetches, so a local server is required.

## Data contract

The UI loads five JSON files from `ui/data/`:

| File | Shape | Source |
| ---- | ----- | ------ |
| `sessions.json` | array of `SessionResult.to_row()` rows (one per session) | the sweep |
| `session_annotations.json` | `{session_id: {turn_idx: {type, fact_idx, claim, truth, tactic, note}}}` | post-hoc Claude classifier |
| `cars.json` | array of `Car.to_json()` (includes `private_facts`) | dataset layer |
| `personas.json` | `{sellers: {id: persona}, buyers: {id: persona}}` | `personas/` |
| `tactics.json` | `{tactic_id: {name, category, description, ...}}` | `tactics/catalog.json` |

`transcripts/<session_id>.jsonl` — one turn per line, matching the `Turn`
dataclass in `used_car_salesman/session.py`. Only sessions referenced in
`session_annotations.json` need a transcript file; only those appear in the
replay view's session picker.

## Regenerate the mock dataset

`generate_mock_data.py` synthesizes a representative sweep (533 sessions
across persona × tactic cells) plus five hand-crafted replayable transcripts.
Used to develop and demo before the real sweep lands.

```bash
python3 ui/generate_mock_data.py
```

## Views

- **Overview** — topline stats, premium curve split by buyer persona, persona
  pair matrix, inspections-vs-premium scatter.
- **Transcript replay** — the iceberg: public listing above the waterline,
  private facts below; facts surface as the conversation plays back, colored
  by mechanism (volunteered / inspection / lie caught). Lies are flagged
  inline with the spoken claim and the truth.
- **Susceptibility** — heatmap with toggleable axes (persona / model / tactic)
  and a tactic-lift bar chart showing per-row premium uplift relative to the
  no-tactic baseline.
- **Methods** — short prose description of the experimental setup.

```
├── style.css               # palette + components
├── js/
│   ├── data.js             # fetch + aggregate
│   ├── overview.js         # overview view
│   ├── heatmap.js          # susceptibility view
│   ├── replay.js           # transcript replay
│   └── app.js              # hash routing + bootstrap
├── data/                   # JSON inputs (regenerated by generate_mock_data.py)
│   ├── sessions.json
│   ├── session_annotations.json
│   ├── cars.json
│   ├── personas.json
│   ├── tactics.json
│   └── transcripts/<session_id>.jsonl
└── generate_mock_data.py
```
